Надёжное сетевое оборудование для скоростного интернета в квартиру или частный дом, сочетающее функции оптического терминала, модема и беспроводного Wi-Fi маршрутизатора.

Эта модель станет оптимальным вариантом для большой квартиры или небольшого офиса.

Технические возможности роутера обеспечивают пользователя мощным интернет-сигналом по оптоволоконному кабелю и хорошей скоростью передачи данных Wi-Fi.

Есть возможность резервного подключения через телефонную линию.
